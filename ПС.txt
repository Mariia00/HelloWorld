Мій перший репозиторій

1.	Що таке Git? 
Git - це набір консольних утиліт, які відстежують і фіксують зміни в файлах. 
2.	В якому статусі можуть перебувати файли при роботі з Git? 
Git має три основних статуси, в яких можуть перебувати ваші файли: збережений у коміті (commited), змінений (modified) та індексований (staged).
3.	Для чого використовується  команда add? 
Команда add використовується для додавання всі файлів до репоиторію.
4.	Для чого використовується  команда status?
Команда status використовується для перевірки чи додані файли .
5.	Для чого використовується  команда commit? 
Команда commit використовується для фіксації стану сховища в певний момент часу, тобто фіксації змін.
6.	Для чого використовується  команда init? 
Команда init використовується для ініціалізації Git у папці проекту.
7.	Що необхідно писати в коментарях комітів? 
Необхідно писати зміни, які відбулися у конкретному комітеті для того, щоб це було зрозуміло.
8.	Як переглянути список комітів? 
Для перегляду списків комітів потрібно скористатися командою "git log".
9.	Для чого використовується  команда checkout?
Команда checkout використовується для перемикання користувача між комітетами проекту.
